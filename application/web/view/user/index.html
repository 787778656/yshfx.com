返佣说明

　　

注册用户通过推广分享个人专属二维码或邀请码发展下级用户可以获取一定的返佣；返佣规则如下：

　　

1、若下级用户选择购买VIP服务，则获取下级用户购买VIP服务费用的20%/月作为佣金；例如：当月若干个下级用户一共购买了100000元服务费，那么将会获取20000元佣金（100000*20%）；注：下级用户自购买跟单服务费的佣金第二天结算到个人中心账户里；提现24小时内到账；

　　

2、若下级用户选择成为永久VIP，则获取下级用户20元/手（交易量）作为佣金；例如：当月若干个下级用户一共交易了1000手，那么将会获取20000元佣金（1000*20）；注：下级用户自开始跟单所产生的交易量佣金第二天结算到个人中心账户里；提现24小时内到账；

　　

3、不限制下级用户选择跟单服务的方式，两种返佣方式独立运行计算互不干扰；
